# IMGD1050---Assignment-5  

This asssignment uses noise and randomness to create a a scene of glowing fireflies in the night sky.  Randomness is used to position both the stars and firelflies and noise is used to complete the visual effects of both the firefly movement and glow.  
The use of the map(), constrain() and sin() functions allow for a more organinc effect.  The map() function is used throughout the whole scene to allow the fireflies to move both left and right as well as up and down.  
Constrain() limits the opacity of the firefly glow and sin() allows the glow to "breathe" as it causes the opacity to oscillate from least to most as described by the constrain function
